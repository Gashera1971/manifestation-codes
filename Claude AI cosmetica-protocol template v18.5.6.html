Handleiding

1. Protocol deels invullen

Vul Fase, %, Ingrediënten, Bereiding en Pr. Info in. Druk op 💾 Opslaan → deelscherm → kies Drive → map "Protocollen om mee te bereiden".

2. In het lab — protocol bereiden

Druk op 📂 Laden → Bladeren → Google Drive → jouw map → tik recept. Vul Batch, V kolom, gtt, Datum aan.

3. Protocol volledig ingevuld

Druk 🔒 Vergrendelen → code invoeren. Dan 💾 Opslaan → Drive → map "Bereide protocollen". Dan 🖨 Afdrukken.

💾 Opslaan naar Drive

1

Druk 💾 Opslaan → deelscherm verschijnt

2

Kies Drive → kies jouw map → Uploaden

3

Drive geeft soms ook een .json.txt → die mag weg

📂 Laden uit Drive

1

Druk 📂 Laden → tik Bladeren

2

Kies Google Drive → jouw map

3

Tik op je recept → laadt direct in!
